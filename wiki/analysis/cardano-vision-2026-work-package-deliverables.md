# Cardano Vision 2026 Work Package Deliverables

**From query:** 2026-05-25  
**Question:** What are the deliverables for each work package in reference to `vote_context/markdown/53_IOR.md`?

## Answer

The vote context for vote 53 summarizes the seven Cardano Vision 2026 work packages by focus area. The underlying IOR proposal’s “Activity of Work” section gives the deliverables below.

| Work package | Deliverables |
|--------------|--------------|
| **WP1: Trust, Security & Reliability Infrastructure** | MEV and mempool partitioning assessment; Linear Leios security and performance analysis; Peras/Leios protocol improvement paper or report; Peras+Leios unification design and analysis; post-quantum cryptography assessment for Cardano; post-quantum VRF design and security proof; quantum-secure Ouroboros security analysis; PQC integration specifications and performance benchmarks; Cardano post-quantum readiness roadmap and migration strategy; node security threat model and key-management analysis; HSM-enabled node implementation for KES/VRF signing with documentation. |
| **WP2: Scalability & Execution Layer** | Leios-compatible fee market design specification; fee-market simulation results and behavioral analysis; data availability requirements and design framework for L2s; Cardano-optimized sharding protocol evaluation and design; paper on Nakamoto blockchains under Byzantine network adversaries and capacity constraints; pub/sub specification and prototype; proof-system design and trade-off analysis; ZK verification infrastructure for Cardano L1 with prototype, documentation, benchmark, and CIP; zk-rollup/Hydra Tail report or paper. |
| **WP3: Developer Platform & User Experience** | EasySM development environment research prototype in Agda; verifying optimizing compiler specification and certification framework; Cavefish protocol paper or report; Cavefish specification and prototype; Cavefish network prototype and benchmarks; decentralized API/indexer specification, prototype, and benchmarks; Babel fees marketplace design and recommended market mechanism; Cardano intent solver market desiderata and wallet/DApp integration exploration; minUTXO alternatives report, trade-off analysis, policy recommendations, and CIP. |
| **WP4: Applications, Adoption & Liquidity** | Mithril bridge security model, analysis, and automated model checking; Mithril verification benchmarks; bridge security and adversarial threat-model analysis; n-party atomic swap prototype and security analysis; agent chain specification, prototype, and integration guide. |
| **WP5: Economic Systems & Incentives** | SPO incentive model assessment and parameter recommendations; multi-resource BFT consensus report or paper and feasibility assessment; incentive mechanisms for restaking and multi-resource consensus; provably secure proof-of-useful-work paper/CPS for state-proof generation; utility-weighted throughput framework and benchmarking analysis. |
| **WP6: Governance, Identity & Social Infrastructure** | Governance incentive models and mechanism assessment with CIP pathway; DAO governance DSL specification, prototype, integration toolkit, and CIP; proof-of-personhood model, ZK identity primitives, and governance prototype; identity primitives and integration specification with prototype, documentation, and CIP. |
| **WP7: Delivery, Dissemination & Partnerships** | Program governance and bi-annual reporting; structured communication outputs such as blogs, explainers, technical summaries, and GitHub repos; Cardano R&D Sessions and ecosystem-facing events; translation of research outputs into implementation artifacts such as CIPs and specs; prioritized integration pathways with infrastructure providers; support for prototype-to-deployment progression with engineering and product teams. |

## Notes

Across the full program, the proposal aggregates these outputs into roughly **38 papers/reports**, **9 Cardano Problem Statements**, **12 prototypes**, and around **5 CIP pathways**. WP7 is listed as in-kind/absorbed by IOG in the budget, but still has reporting, communication, and commercialization deliverables.
