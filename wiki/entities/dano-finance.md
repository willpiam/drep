# Dano Finance

**Dano Finance** is a Cardano DeFi protocol team (with named contributor **fallen-icarus**) that proposes **[[proposals/global-order-book-defi-kernel|Global Order Book / DeFi Kernel]]** treasury funding. Public site: [dano.finance](https://dano.finance). DeFi Kernel registry work references [defikernel.org](https://defikernel.org) and a GitHub registry repo.

**Bottom line:** We know the **orgs** (Dano/Danogo + Minswap admin) and **one named individual** tied to the DeFi Kernel/order-book work (**fallen-icarus / Rusty**). Dano’s own About page does **not** publish a named founders roster, and the on-chain proposal metadata does not attestation-sign named authors.
## Track record (self-reported in proposal)

From [[sources/global-order-book-connect-cardano-defi]]:

- Operating metrics at submission: ~**$18M TVL** and **10,000+ on-chain transactions**.
- Product surface claimed: **lending**, **concentrated liquidity**, **oracle aggregation**, **fixed and flexible borrowing**, and **composable transaction flows**.
- Framing: experience shipping production DeFi on Cardano’s eUTxO model (liquidity bootstrapping, protocol safety, tx composition, onboarding, measurable impact).
- Roadmap already aimed at synthetics, leverage, refinancing, and single-transaction execution — presented as aligned with order-book / financial-intent coordination.

**Gap:** The wiki does not yet have an independent product-by-product inventory or third-party verification of TVL/tx counts beyond the proposer’s statements and linked public sites.

## Reputation snapshot (external, 2026-07-14)

**Overall: mid-tier Cardano DeFi operator — real product footprint, modest brand recognition; not a top-tier household name.**

| Signal | Observation |
|--------|-------------|
| Presence | Live since ~Oct 2023 (Bond DEX → lending/leverage/yield); listed on Cardano.org apps, Adastack, DefiLlama |
| Size | DefiLlama ~**$13M TVL** (proposal self-reported ~$18M); ~$4.8M 30d DEX volume; low annualized fees (~$6k) |
| Security | Anastasia Labs audits cited for bond/lending/leverage/oracles/pools ([docs](https://docs.dano.finance)); DefiLlama marks Audits: Yes |
| Social | [@DanoFinance](https://x.com/DanoFinance) — ~3.7k followers, ~9.6k posts since May 2023; active but mid-small for Cardano DeFi |
| Transparency drag | No public named team on About / CardanoCube; CIP authors empty; CardanoCube project rating “No votes yet” |
| Co-proposer signal | **fallen-icarus / Rusty** has stronger *technical-niche* reputation (Cardano-Swaps, Dev Portal / Cardano360 interviews) than Dano has *product-brand* recognition |

Judgment for DRep diligence: credible enough as a shipping protocol (TVL + audits + longevity), **not** strong on personal accountability / brand prestige compared with Minswap, Liqwid, IOG, etc.
