# 5am.earth Trust Layer Targeting Vision 2030 KPIs

**Type:** Treasury withdrawal (real-world adoption / trust infrastructure)  
**Proposer:** [[entities/5am-earth-foundation|5am.earth Foundation]] (pre-M1 co-promoters: Elk GMBH, HashPoint Consulting Sarl)  
**Administrator:** [[entities/intersect|Intersect]] (TRSC/PSSC, standard terms)  
**Source:** [[sources/5am-earth-trust-layer-vision-2030-kpis]]

The Foundation requests **₳10,000,000 (hard cap)** over **18 months** to scale a Cardano-anchored **[[concepts/agricultural-trust-layer|agricultural trust layer]]** — shared verified farmer/farm/crop infrastructure that multiple applications reuse ("verify once, use many times").

## Trust layer architecture

Three layers:

1. **Source layer** — AEs, farmers, land/crop records, satellite observations, credentials, application events (traceability, credit, compliance).
2. **Trust layer on Cardano** — verifiable claims, on-chain proofs, privacy/permissions, audit trail.
3. **Application layer** — partners connect via APIs, credentials, selective disclosure.

Cardano components cited as **operational today** (not greenfield): AIQuant/Anastasia Labs smart contracts; DigiFarm satellite oracle; Veridian DID; Andamio credentials; NMKR tokenisation.
