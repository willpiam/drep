# Project Swaminathan

**Project Swaminathan** is a Cardano Mainnet agricultural onboarding programme operated by [[entities/syngenta-foundation-india|Syngenta Foundation India]]. It is the live operational baseline for the [[concepts/agricultural-trust-layer|agricultural trust layer]] scale-up in [[proposals/5am-earth-trust-layer-vision-2030-kpis]].

## On-chain operations

| Metric | Value (as of proposal) |
|--------|------------------------|
| Cumulative registrations | 10,500 farmers/farms (5 May 2026) |
| Registration rate | ~500/day |
| Geography | Six Amravati districts, Maharashtra |
| Identity | Veridian self-sovereign identity per farmer |
| Farm records | On-chain verified land and crop details |

**Pilot (Jan–Mar 2026):** 1,056 farms validated across three phases at 100% blockchain success rate. A same-day critical patch during Phase 3 with zero downtime demonstrated field-to-Mainnet resilience. Scale-up began 1 April 2026.

## Technical stack

Smart contract integration (AIQuant, Anastasia Labs), DigiFarm satellite oracle, Veridian DID — same components the trust layer proposal locks at M1 procurement.

## Funding history

- **Catalyst Fund 13** (project ID 1300178) — initial baseline
- **Treasury scale-up:** [[proposals/5am-earth-trust-layer-vision-2030-kpis]] — ₳10M, 18 months, multi-country expansion
